# 通信协议

## 战斗外通信

### 由客户端发送

#### `init_battle`

参数：无

初始化战斗，必须最先发送。

#### `start_battle`

参数：无

开始战斗，必须最后发送。

#### `add_character`

参数：

- `dict record`：角色设置

添加一个角色到队伍末尾。

#### `setup_monsters`

参数：

- `dict record`：所有怪物的设置

设置战斗使用的怪物的信息。

#### `setup_random`

参数：

- `dict config`：随机生成器的设置

设置随机生成器的种子，或每次需要随机时询问客户端。

#### `set_initial_state`

参数：

- `uuid target`：目标
- `dict state`：初始状态

设置目标的初始状态。

### ASK

由服务端发送，客户端回复。

#### `ultimate`

发送参数：无

接收参数：

- `uuid character`：使用终结技的角色
- （可选）`uuid target`：终结技的技能目标

询问使用角色的终结技。

错误：

- `invalid_message_type`：消息类型或名称不正确。
- `invalid_message`：消息参数格式不正确。
- `target_not_found`：`character`对应的目标不存在。
- `target_not_character`：`character`对应的目标不是角色。
- `ultimate_activated`：技能已经激活。
- `not_enough_energy`：能量不足。
- `ultimate_not_available`：技能当前无法释放。
- `bad_target`：技能目标不合法。
- `internal_error`：内部错误。
